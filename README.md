
# Endo-NeRF++: Uncertainty-Aware Sampling and Multi-Resolution Encoding for Dynamic Surgical Scene Reconstruction
<p align="center">
  <img src="assassets/Research Slides.png" width="800"/>
</p>


This repository presents EndoNeRF++, a scalable framework for dynamic surgical scene reconstruction. It integrates multi-resolution hash encoding for efficient representation and uncertainty estimation for reliability, enabling accurate and robust modeling of deformable surgical environments.
